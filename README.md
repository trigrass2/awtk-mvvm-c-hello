# awtk-mvvm-c-hello

awtk mvvm 完整示例

## 准备

1.获取awtk并编译

```
git clone https://github.com/zlgopen/awtk.git
cd awtk; scons; cd -
```

2.获取awtk-mvvm-c-hello并编译
```
git clone https://github.com/zlgopen/awtk-mvvm-c-hello.git
cd awtk-mvvm-c-hello; scons
```

## 运行

```
./bin/awtk_mvvm_app
```


> 本文以Linux/MacOS为例，Windows可能会微妙差异，请酌情处理。
